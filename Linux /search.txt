# Content
grep "some_text" -ir
-r or -R is recursive
-n is line number
-w stands match the whole word
-l (lower-case L) can be added to just give the file name of matching files
-i stands for ignore case (optional in your case)
/ stands for starting at the root of your machine.

ack "some_text"


Example: 
- search for files in the last 3 days and the result is written to a file;

find . -mtime -3 -type f -printf '%TY-%Tm-%Td %TT %p\n' | sort -r > log.txt
